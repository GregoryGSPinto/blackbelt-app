# Docs Index

Documentação viva do produto autenticado BlackBelt.

## Taxonomia

- `architecture/`: topologia de domínios, boundary design e modelo repo/app/api/mobile
- `operations/`: segurança, integrações, SDK e operação técnica
- `release/`: deploy, gates de release, metadata e runbooks
- `store/`: requisitos e artefatos de App Store / Play Store
- `audits/`: auditorias, relatórios, testes reais e documentos removidos da raiz
- `archive/`: artefatos históricos preservados
- `setup/`: onboarding técnico e convenções
- `product/`: roadmap e maturidade
